## 🚀 Getting Started

### Prerequisites
```bash
Node.js 18+
npm or yarn
```

### Installation
```bash
# Clone repository
git clone <repo-url>

# Navigate to frontend
cd quizsync-frontend

# Install dependencies
npm install

# Run development server
npm run dev

# Open browser
http://localhost:3000
```

### Build for Production
```bash
# Create production build
npm run build

# Start production server
npm start
```
